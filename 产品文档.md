## 产品定位

**产品名**：KnowledgeAI（AI知识助手SaaS）

**一句话介绍**：上传文档 → AI构建知识库 → 团队智能问答 → 自动生成调研报告

**整合的技术**：

| 技术领域  | 在产品中的体现                     |
| --------- | ---------------------------------- |
| 全栈开发  | Next.js前后端 + PostgreSQL + Redis |
| AI/ML应用 | RAG问答 + Agent调研 + 文档处理     |
| 云原生    | Docker部署 + K8s编排 + CI/CD       |
| 系统设计  | 多租户 + 微服务 + 消息队列         |
| 自动化    | GitHub Actions + 自动测试          |
| 业务理解  | 订阅计费 + 用量监控                |
| 安全合规  | 数据隔离 + GDPR合规                |

- 流式响应（Server-Sent Events）
- 文件上传处理（multer / formidable）
- 异步任务队列（BullMQ + Redis）

1. **RAG 引擎全异步化**：嵌入/检索/生成均支持 OpenAI LLM，配置 `OPENAI_API_KEY` 即从本地抽取式切换为真实 LLM 流式生成
2. **Provider 适配层**：6 个适配层实现「配置即切换」，未配置时优雅降级到演示模式，业务代码零改动
3. **SSE 流式**：Chat（逐 token 问答）与 Agent（4 步执行进度）均使用 Server-Sent Events 实时推送
4. **RBAC 权限**：4 角色 × 10 能力矩阵 + 共享知识库访问级别 + 审计日志
5. **安全合规**：限流中间件 + JWT 认证 + 2FA + GDPR 数据导出 + 安全响应头 + 非 root Docker
